분산 강화학습 논문(DeepMind IMPALA) 구현


발표자 : 김정주 ( )

시간 : 오후 5시 (20분)

난이도 : 어려움

키워드 : 강화학습, 분산, 구현

내용

최근 강화학습은 학습 알고리즘 고도화와, 분산 처리라는 두 방향으로 발전하고 있습니다. 이 발표는 딥마인드의 분산 강화학습 모델인 IMPALA를 소개하고, 그 구현 방법을 살펴봅니다.

대상 청중

분산 강화학습의 이해와 구현에 관심있는 분들.

발표 이해에 도움 되는 사전 지식

기본적인 강화학습 알고리즘 지식.

연사 소개

오랫동안 게임 개발자로 일하다, 현재 (주)웹젠에서 빅데이터와 기계학습을 사용해 게임 데이터 분석 업무를 맡고 있습니다. 부가적으로 딥러닝과 강화학습 등의 주제로 인공지능 연구를 계속하고 있습니다.
