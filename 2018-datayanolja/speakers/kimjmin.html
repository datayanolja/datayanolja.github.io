Elasticsearch 검색엔진의 한글 형태소 분석기 nori


발표자 : 김종민 ( )

시간 : 20분

난이도 : 그럭저럭 쉬움

키워드 : Elasticsearch, Elastic, 형태소 분석, 한글, 그놈의빅데이터

내용

인기있는 검색엔진 Elasticsearch 에서 텍스트 처리하는 방법에 대해 설명합니다. 그리고 새로 출시된 nori 한글 형태소 분석기에 대해서 알아봅니다.

대상 청중

검색엔진, 검색 기술에 대해 관심 있는 분.

발표 이해에 도움 되는 사전 지식

Elasticsearch 구글링. 작년 데놀 발표 (중복 내용 있습니다)

연사 소개

Elastic 사의 Community Engineer로 Elasticsearch 기술 커뮤니티를 운영하고 다양한 저술, 강연, 발표 등의 활동을 하며 개발자들과 소통하고 있습니다.
