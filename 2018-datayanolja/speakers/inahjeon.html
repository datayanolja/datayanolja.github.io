데이터 과학자의 코딩없는 서비스 로그 분석 인프라 구축 삽질기


발표자 : 전인아 ( )

시간 : TBD

난이도 : 쉬움

키워드 : 앱 로그 분석, Google Cloud Platform, Firebase, BigQuery

내용

서비스를 잘 만들기 위해서는 서비스를 사용하는 고객에 대해 잘 이해하는 것이 가장 중요합니다.
이를 위해 앱 서비스 로그 분석을 본격적으로 시작하게 되었고, 데이터 분석 인프라에 관한 지식이 없고 큰 개발 리소스를 쏟을 수 없었던 상황에서, GCP에서 쉽게 도입해 볼 수 있는 서비스들을 엮어서 로그 분석 인프라를 구성해보기로 결정했습니다.
그 과정에서 겪었던 어려움들과, 로그 분석을 더 쉽게, 효율화하기 위해 어떤 삽질을 했었는지, 문제를 어떻게 해결했었는 지에 대한 경험담을 공유합니다.

대상 청중

- 앱 로그 데이터 분석을 시작하게 된 데이터 분석가
- 로그 분석 인프라를 따로 구성할만한 엔지니어링 리소스가 없는 데이터팀의 분석가

발표 이해에 도움 되는 사전 지식

크게 필요하지 않음. GCP, Firebase, BigQuery

연사 소개

데이터마이닝 연구실을 졸업하고 모 대기업 연구소에서 일하다가, 살아서 팔딱팔딱 뛰는(?) "Real-world" 데이터를 다루고 싶어 연구소를 뛰쳐나오게 되었고, 자산관리 서비스를 만들고 있는 레이니스트에 합류하여 데이터 분석가로 일하고 있습니다.
현재 뱅크샐러드 서비스에 필요한 데이터 분석, 머신러닝 서비스 개발을 담당하고 있습니다.
데이터로부터 지금보다 더 많은 가치를 만들고, 더 멋진 서비스를 만드는 것을 꿈꾸고 있습니다.
