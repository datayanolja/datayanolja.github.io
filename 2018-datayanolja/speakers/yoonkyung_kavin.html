아이돌 음악으로 분석하는 아이돌 시장의 젠더불균형


발표자 : 김윤경 ( )

권가빈 ( )

시간 : 18:30 - 19:00

난이도 : 쉬움

키워드 : 아이돌, 가사, 젠더불균형, 텍스트마이닝, 데이터시각화

내용

아이돌 가사 속 외양 묘사, 성적 대상화에 대한 문제의식에서 출발해 아이돌 가사와 산업 속 젠더 불균형을 분석한 프로젝트입니다.
멜론을 크롤링하여 곡, 가수 정보를 수집하여 가사 속 성별 외양 묘사 빈도, 종사자 비율, 여자/남자 멤버의 프로덕션 참여 비율 등에 대한 가설을 세우고 비교 분석하였습니다.
이를 통해 한국 아이돌 산업의 젠더 감수성 결여를 데이터로 풀어내 보았습니다.

대상 청중

데이터 분석에 관심이 있지만 내가 할 수 있을까? 라는 막연한 걱정이 드는 비전공자 분들,
자연어의 모호성에 흠뻑 빠져보고 싶으신 분들,
d3를 활용한 데이터 시각화에 관심이 있으신 분들,
데이터를 통해 탐구해보고 싶은 문제가 있으신 분들! 들으시면 분명 재미있으실 겁니다.

발표 이해에 도움 되는 사전 지식

자연어처리에 대한 호기심, 파이썬으로 무얼 할 수 있는가, K-POP 아이돌 문화에 대한 이해

연사 소개

김윤경 : 이제 막 데잇걸즈를 수료하고 새로운 시작점에 선 주니어 데이터분석가입니다.
자연어처리(NLP)와 데이터 저널리즘에 관심이 많습니다.
데이터로 세상을 바꾸고자 하는 큰 꿈이 있습니다.

권가빈 : 데잇걸즈 2기 수료를 시작으로 데이터분석의 매력에 흠뻑 빠졌습니다.
앞으로 계속 성장해 나아가면서 이후 데이터사이언티스트로서 활약하고 싶습니다:)
