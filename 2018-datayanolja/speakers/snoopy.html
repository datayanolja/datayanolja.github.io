발표자 : 장현석 ( )

시간 : TBD

난이도 : 쉬움

키워드 : 미국, 뉴욕, 미디어, 엔터테인먼트, 데이터 사이언스, 참관기

내용

[Data Science Salon]은 미국 현지의 데이터 사이언티스트들이 서로 모여 프로젝트를 공유하고 새로운 경험을 나누는 자리입니다. 미국판 [데이터야 놀자!]같은 컨퍼런스이랄까요? 9월27일 뉴욕에서 열린 미디어업계 관련 컨퍼런스를 참관한 후기를 나누고자 발표를 준비하였습니다!

대상 청중

데이터 사이언스가 미디어를 어떻게 변화시키고 있는지 궁금한 분들

발표 이해에 도움 되는 사전 지식

data science salon에 대한 구글링

연사 소개

10여년간 게임업계에서 일하다가 컨텐츠 데이터마이닝 전문 업체 (주)빅레이더를 창업하였습니다. 영화, 웹툰, 게임 컨텐츠에 대한 데이터마이닝을 꾸준히 진행 중이며, 2018년 서울신용보증재단에서 주최 상권분석 빅데이터 경진대회 최우수상 수상하였습니다.
