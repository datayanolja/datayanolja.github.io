회사는 어떤 사람을 데이터 분석가로 채용하고 싶어하는 것일까?


발표자 : 윤선미 ( )

시간 : TBD

난이도 : 쉬움

키워드 : 데이터 분석가, 직무설명서, Job description, 인사데이터, 구직, 취뽀(?), Pre-trained word vectors, TF-IDF

내용

'회사는 어떤 사람을 데이터 분석가로 채용하는 것일까?'는 질문을 항상 가지고 있었습니다. 좀 더 질문을 구체적으로 해볼게요. 회사는 어떤 업무를 하기위해, 어떤 역량이 있는 사람을 데이터 분석가로 채용하는 것일까요? 이런 질문들에 데이터 분석가다운 답을 하기 위해 원티드, 로켓펀치, 사람인 등 국내의 굵직한 구인구직 사이트들의 데이터 분석가 직무설명서를 수집, 분석 해보겠습니다.

                                    

발표 자료

                                    

                                            Uploaded on Speaker Deck                                     

대상 청중

데이터 분석 직군(데이터 분석가, 데이터 사이언티스트, 비즈니스 애널리스트 등)으로 커리어 디자인을 하고있는 모든 사람

발표 이해에 도움 되는 사전 지식

'데이터 분석가', '데이터 사이언티스트'로 원티드, 로켓펀치 등 구인구직 사이트 검색해보고 오시면 더 재미있게 발표를 들으실 수 있습니다.

연사 소개

쿠팡의 배송시스템에서 2년 동안 데이터 분석을 하고 있습니다. 울산과학기술원에서 경영공학(계량금융) 석사과정을 했습니다. 레스큐 스쿠버 다이버, SSI1 프리다이버입니다.
